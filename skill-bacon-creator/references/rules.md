## Rule 1
**If** the target API returns HTTP 401 on unauthenticated request,
**then** fallback to search_web scraping before declaring the endpoint inaccessible.
**If that fails,** log the auth wall in observations.md and flag as Known Gap in SKILL.md.
**Source:** Observation 2 — auth wall hit on `/me` endpoint, no token provided

## Rule 2
**If** `search_web` returns an empty snippet on first query,
**then** validate with `exec` + curl before assuming no data exists.
**Source:** Experiment 1 — empty result on first query, curl returned valid JSON response

## Rule 3
**If** a rate limit (HTTP 429) is hit on first attempt,
**then** back off 2 seconds and retry once only. Do not retry more than once.
**Source:** Observation 3 — 429 returned on high-frequency probe, single retry succeeded
