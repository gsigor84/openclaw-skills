---

## What belongs here

- Verbatim tool call outputs (truncated at 500 chars if needed)
- HTTP status codes and raw response bodies
- Error messages, auth walls, rate limit responses
- exec command outputs
- Idol flags surfaced during observation

## What does NOT belong here

- Summaries or paraphrases of tool outputs
- Assumed or invented results
- Blog post content or documentation quotes
- Anything not produced by a live tool call in this session
